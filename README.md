# navicode

A custom coding assistant CLI tool using OpenRouter's API.

## Features

- Chat interface with customizable AI model
- Tool system (e.g., reading, writing...)
- Colored output

## Requirements

- Python 3.x
- OpenRouter API key

## Usage

Run `OPENROUTER_API_KEY=<your_key> MODEL=<your_model> DEBUG=0 python navicode.py` from your project directory.
